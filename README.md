# ConcurrencyAsync

Application de démonstration WinForms conçue pour explorer et comprendre la programmation asynchrone et concurrente en C#.

## Fonctionnalités

*   Téléchargement de fichiers à partir d'une URI.
*   Interface utilisateur réactive pendant les opérations de téléchargement grâce à l'asynchronisme.
*   Indicateur de progression visuel.
*   Notification de fin de téléchargement.
*   Dialogue "Enregistrer sous" pour choisir le répertoire de destination.

Ce projet met en évidence comment effectuer des opérations I/O intensives sans bloquer l'interface utilisateur.
